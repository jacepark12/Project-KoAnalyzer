KoAnalyzer 기능

웹 크롤링

구글을 통해 원하는 키워드를 검색한 후, 검색된 결과들을 파일로 저장합니다.

정규화 & 형태소 분석

한국어 자연어를 정규화한 후, 형태소 분석을 진행하여 결과를 JSON으로 반환합니다.

Document 분석

한국어로 작성된 문서에서 감성분석과 밀집도 분석을 기반으로 키워드 추출, 형태소 분석을 진행합니다.

문장 분석

한국어로 작성된 문장에서 정규화 처리를 진행한 후, 형태소 분석을 진행하여 JSON형태로 반환합니다
